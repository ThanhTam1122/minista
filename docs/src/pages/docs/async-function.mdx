minista の JSX で使用できる非同期関数です。

## getStaticData

ページ生成前、非同期に処理した結果から `props` と `paths` をページテンプレートに渡せます。主に [Fetch](/docs/fetch) で CMS の API からデータを取得しページ生成する目的で使います。

## StaticData

TypeScript で `getStaticData` に型を付与する場合、アロー関数には `GetStaticData` を、function の戻り値には `Promise<StaticData>` を使います。
